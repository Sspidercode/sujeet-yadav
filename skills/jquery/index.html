jQuery

Legacy DOM utilities, animations, and AJAX helpers for older codebases and quick prototypes. Below is a comprehensive guide to key jQuery concepts with detailed explanations and examples. (Assumes jQuery is loaded via CDN.)

Basic Example: Document Ready

This ensures code runs after the DOM is ready, a common jQuery idiom to avoid timing issues.

$(function(){ console.log('Ready'); });
Open example.txt
1. Selectors

jQuery uses CSS-like selectors to find elements. Basic: tag, class (.class), ID (#id). Advanced: attribute [attr=value], pseudo :even, :odd, :first, etc. Returns a jQuery object for chaining.

// Basic selectors
$('h1').css('color', 'red');  // All h1 tags
$('.title-v2').hide();        // Class selector
$('#year').text('2025');      // ID selector

// Advanced
$('[href]').attr('target', '_blank');  // All links
$('li:even').addClass('highlight');    // Even list items
2. DOM Manipulation

Methods to add, remove, or modify elements/content. Includes append, prepend, after, before, remove, empty, html, text.

// Add content
$('body').append('
New paragraph
');
$('.panel-v2').prepend('Header: ');

// Modify
$('.title-v2').html('Updated Title');
$('img').attr('src', 'new-icon.svg');

// Remove
$('.old-class').remove();
$('#temp').empty();
3. Events

Handle user interactions with on(), off(), one(), trigger(). Supports event delegation for dynamic elements. Common events: click, hover, submit, keydown.

// Click event
$('button').on('click', function(e) {
$(this).toggleClass('active');
e.preventDefault();
});

// Hover (mouseenter, mouseleave)
$('.icon').hover(
function() { $(this).fadeIn(300); },
function() { $(this).fadeOut(300); }
);

// Form submit
$('form').on('submit', function(e) {
e.preventDefault();
alert('Submitted!');
});

// Remove listener
$('button').off('click');
4. Effects and Animations

Built-in methods for show/hide, fade, slide, animate(). animate() allows custom CSS properties with duration, easing, callback.

// Basic effects
$('.panel-v2').hide().show(500);  // Hide then show over 500ms
$('img').fadeIn(1000).fadeOut(1000);

// Slide
$('.content').slideToggle('slow');

// Custom animate
$('div').animate({
opacity: 0.5,
left: '+=50px',
height: 'toggle'
}, 1000, 'swing', function() {
console.log('Animation complete');
});
5. AJAX

Simplified HTTP requests with $.get(), $.post(), $.ajax(). Handles success, error, complete callbacks. Supports JSON, XML, etc.

// GET request
$.get('https://api.example.com/data', function(data) {
console.log(data);
});

// POST
$.post('url', { name: 'Sujeet' }, function(response) {
$('#result').html(response);
}, 'json');

// Full AJAX
$.ajax({
url: 'endpoint',
method: 'POST',
data: { key: 'value' },
dataType: 'json',
success: function(data) { console.log(data); },
error: function(xhr, status) { console.error('Error:', status); }
});
6. Traversing

Navigate the DOM tree: parent(), children(), siblings(), find(), next(), prev(), closest(). Filters with .filter(), .not().

// Up/down
$('li').parent().addClass('parent-highlight');
$('.child').children().css('font-weight', 'bold');

// Sideways
$('li:first').nextAll().hide();

// Descendants
$('.container').find('p').toggle();

// Closest ancestor
$('span').closest('.panel-v2').slideDown();

// Filter
$('li').filter(':even').css('color', 'blue');
7. Attributes and CSS

Manipulate attributes with attr(), removeAttr(), prop(), data(). CSS with css() for styles, addClass(), removeClass(), toggleClass().

// Attributes
$('img').attr('alt', 'New alt text');
$('a').removeAttr('target');

// Properties (e.g., checked, disabled)
$('input[type="checkbox"]').prop('checked', true);

// CSS
$('.title-v2').css({ 'color': 'red', 'font-size': '2em' });

// Classes
$('body').addClass('theme-light').removeClass('theme-dark');
$('button').toggleClass('btn-primary');
8. Forms

Handle form data with serialize(), val(), each(). Validation can be custom; jQuery UI for advanced.

// Get/set value
$('input[name="email"]').val('sujeet@example.com');

// Serialize form
let formData = $('form').serialize();
console.log(formData);  // name=John&email=john@example.com

// Iterate inputs
$('form input').each(function() {
if ($(this).val() === '') {
  $(this).addClass('error');
}
});

// Reset form
$('form')[0].reset();  // Or $('form').trigger('reset');
9. Utilities

Helper functions like $.each(), $.map(), $.extend(), $.isArray(), $.trim(). Useful for data manipulation without DOM.

// Each loop
$.each(['JS', 'jQuery'], function(index, value) {
console.log(index + ': ' + value);
});

// Map
let doubled = $.map([1, 2, 3], function(n) { return n * 2; });
console.log(doubled);  // [2, 4, 6]

// Extend (merge objects)
let defaults = { theme: 'dark' };
let options = $.extend(defaults, { color: 'blue' });
console.log(options);  // { theme: 'dark', color: 'blue' }

// Trim
let str = $.trim('  hello  ');  // 'hello'
10. Plugins and Best Practices

jQuery ecosystem includes plugins (e.g., via $.fn.extend). For legacy: Use noConflict() with other libs. Best practices: Chain methods, cache selectors, migrate to vanilla JS where possible for performance.

// Simple plugin
$.fn.highlight = function() {
return this.css('background', 'yellow');
};
$('.text').highlight();

// noConflict
let $j = jQuery.noConflict();
$j('p').hide();

// Chaining
$('#content').find('p').addClass('highlight').fadeIn(500);

// Cache
let $title = $('.title-v2');
$title.html('Updated').css('color', 'green');

This covers core jQuery concepts. For modern apps, prefer vanilla JS or frameworks like React. Explore jQuery docs for version-specific details.
