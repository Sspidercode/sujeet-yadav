1. Syntax and Variables

PHP scripts start with <?php and end with ?>. Variables use $ prefix; types include scalar (int, string, bool) and compound (array, object). Strict typing with declare(strict_types=1).

<?php
          declare(strict_types=1);
          
          $name = 'Sujeet';  // String
          $age = 30;         // Integer
          $isDev = true;     // Boolean
          
          echo "Name: $name, Age: $age";  // Interpolation
          
          // Type check
          var_dump($name);  // string(6) "Sujeet"
2. Control Structures

Conditionals (if/else, switch) and loops (for, while, foreach) control flow. Use === for strict comparison.

<?php
          $score = 85;
          
          if ($score >= 90) {
              echo 'A';
          } elseif ($score >= 80) {
              echo 'B';
          } else {
              echo 'C';
          }
          
          // Switch
          switch ($score) {
              case 90:
                  echo 'Excellent';
                  break;
              default:
                  echo 'Good';
          }
          
          // Foreach loop
          $skills = ['PHP', 'JS', 'MySQL'];
          foreach ($skills as $skill) {
              echo "$skill ";
          }
3. Functions

Functions encapsulate code; support parameters, return types, defaults. Arrow functions (PHP 7.4+) for closures.

<?php
          function greet(string $name, int $age = 25): string {
              return "Hello, $name! You are $age years old.";
          }
          
          echo greet('Sujeet', 30);
          
          // Arrow function
          $add = fn($a, $b) => $a + $b;
          echo $add(5, 3);  // 8
          
          // Variable functions
          $say = 'greet';
          $say('World');  // Calls greet('World')
4. Arrays

Arrays are dynamic; indexed/associative. Functions like array_map, array_filter for manipulation. PHP 8.1+ enums for typed arrays.

<?php
          $fruits = ['apple', 'banana', 'cherry'];  // Indexed
          $person = ['name' => 'Sujeet', 'age' => 30];  // Associative
          
          // Access
          echo $fruits[1];  // banana
          echo $person['name'];  // Sujeet
          
          // Functions
          $upper = array_map('strtoupper', $fruits);
          print_r($upper);  // ['APPLE', 'BANANA', 'CHERRY']
          
          // Filter
          $evenAges = array_filter([1,2,3,4], fn($n) => $n % 2 === 0);
          print_r($evenAges);  // [2 => 2, 4 => 4]
5. Object-Oriented Programming (OOP)

Classes define objects with properties/methods. Support inheritance, interfaces, traits, abstract classes. PHP 8+ attributes for metadata.

<?php
          class Person {
              public string $name;
              protected int $age;
              
              public function __construct(string $name, int $age) {
                  $this->name = $name;
                  $this->age = $age;
              }
              
              public function greet(): string {
                  return "Hi, I'm {$this->name}";
              }
          }
          
          class Developer extends Person {
              public function __construct(string $name, int $age) {
                  parent::__construct($name, $age);
              }
              
              public function code(): string {
                  return $this->greet() . ' and I code in PHP!';
              }
          }
          
          $dev = new Developer('Sujeet', 30);
          echo $dev->code();
          
          // Trait
          trait Loggable {
              public function log(string $msg): void {
                  echo "Log: $msg";
              }
          }
6. Error Handling and Exceptions

Use try/catch for exceptions; set_error_handler for errors. PHP 8+ match for pattern matching in handling.

<?php
          function divide(float $a, float $b): float {
              if ($b === 0) {
                  throw new InvalidArgumentException('Division by zero');
              }
              return $a / $b;
          }
          
          try {
              echo divide(10, 0);
          } catch (InvalidArgumentException $e) {
              echo "Error: " . $e->getMessage();
          } finally {
              echo " Cleanup done";
          }
          
          // Match (PHP 8+)
          $result = match($status) {
              200 => 'OK',
              404 => 'Not Found',
              default => 'Error'
          };
7. Database Interactions with PDO

PDO for secure DB access; prepared statements prevent SQL injection. Supports multiple drivers (MySQL, PostgreSQL).

<?php
          $dsn = 'mysql:host=localhost;dbname=test';
          $pdo = new PDO($dsn, 'user', 'pass', [PDO::ATTR_ERRMODE => PDO::ERRMODE_EXCEPTION]);
          
          // Prepared query
          $stmt = $pdo->prepare('SELECT * FROM users WHERE id = ?');
          $stmt->execute([1]);
          $user = $stmt->fetch(PDO::FETCH_ASSOC);
          print_r($user);
          
          // Insert
          $stmt = $pdo->prepare('INSERT INTO users (name) VALUES (?)');
          $stmt->execute(['Sujeet']);
          echo $pdo->lastInsertId();
8. Sessions and Cookies

Sessions store server-side data; cookies client-side. Use session_start(); setcookie() for HTTP-only security.

<?php
          session_start();
          $_SESSION['user'] = 'Sujeet';
          
          if (isset($_SESSION['user'])) {
              echo "Welcome, {$_SESSION['user']}!";
          }
          
          // Cookie
          setcookie('theme', 'dark', time() + 3600, '/', '', true, true);  // HTTP-only, secure
          
          // Read cookie
          if (isset($_COOKIE['theme'])) {
              echo "Theme: {$_COOKIE['theme']}";
          }
          
          // Destroy session
          session_destroy();
9. Composer and Dependencies

Composer manages packages; autoload PSR-4 classes. composer.json for requirements; dump-autoload for updates.

// composer.json
          {
              "require": {
                  "monolog/monolog": "^3.0"
              },
              "autoload": {
                  "psr-4": {
                      "App\\": "src/"
                  }
              }
          }
          
          // Install: composer install
          
          // Usage
          require 'vendor/autoload.php';
          use Monolog\Logger;
          use Monolog\Handler\StreamHandler;
          
          $log = new Logger('name');
          $log->pushHandler(new StreamHandler('app.log', Logger::WARNING));
          $log->warning('Foo!');
10. Modern PHP 8+ Features

PHP 8+ includes JIT, attributes, named arguments, union types. Enhances performance and expressiveness.

<?php
          // Named arguments
          function info(string $name, int $age = 0): void {
              echo "$name is $age years old";
          }
          info(age: 30, name: 'Sujeet');  // Order independent
          
          // Union types
          function process(mixed $data): string|bool {
              return is_string($data) ? strtoupper($data) : false;
          }
          
          // Attributes (PHP 8)
          #[Attribute]
          class Loggable {}
          
          #[Loggable]
          class MyClass {
              // ...
          }
          
          // Readonly properties (PHP 8.1)
          class Config {
              public function __construct(public readonly string $host) {}
          }

This covers core PHP concepts. For frameworks like Laravel/Symfony, explore their docs. Follow PSR standards for clean, maintainable code. PHP 8.3+ recommended for production.
